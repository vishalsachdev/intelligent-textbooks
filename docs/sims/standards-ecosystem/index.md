## Standards Categories

### Core Data Standards (Blue)
- **xAPI**: Tracks learning activities as statements
- **LRS**: Stores learning records

### IEEE Standards (Purple)
- **IEEE LTSC**: Learning Technology Standards Committee
- **IEEE LOM**: Learning Object Metadata for content description

### IMS Global Standards (Teal)
- **LTI**: Tool interoperability with LMS
- **Caliper**: Analytics standard
- **CLR**: Portable credentials

### Regulatory Requirements (Red)
- **FERPA**: US student privacy
- **GDPR**: EU data protection

## How They Connect

1. **Intelligent textbooks** generate learning data using **xAPI**
2. Data is stored in an **LRS** (institution or student-controlled)
3. **LTI** connects textbooks to **LMS** platforms
4. **IEEE LOM** describes content for discovery
5. All Level 3+ systems must comply with **FERPA/GDPR**
